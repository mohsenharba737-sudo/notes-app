📒 تطبيق الملاحظات – Notes App
تطبيق ويب حديث وخفيف لإدارة الملاحظات، مصمم بلمسة جمالية احترافية، ويعمل بالكامل داخل المتصفح دون أي خادم خارجي.
يوفّر تجربة كتابة سلسة، حفظ تلقائي، بحث فوري، دعم للوضع الليلي، وتصدير/استيراد الملاحظات.

✨ المميزات
📝 إنشاء وتعديل الملاحظات
إضافة ملاحظة جديدة بسهولة

تعديل الملاحظات الموجودة

حفظ تلقائي في LocalStorage

دعم اختصار Enter للحفظ السريع

🔍 بحث فوري
البحث في العنوان والمحتوى والتصنيف

تحديث النتائج مباشرة أثناء الكتابة

🎨 تصميم جميل ومتجاوب
واجهة حديثة بألوان هادئة

دعم كامل للهواتف والأجهزة اللوحية

تأثيرات وحركات خفيفة تزيد من جمال التجربة

🌙 الوضع الليلي (Dark Mode)
تبديل فوري بين الوضعين

حفظ تفضيل المستخدم في LocalStorage

📤 تصدير الملاحظات
تصدير جميع الملاحظات إلى ملف JSON

مناسب للنسخ الاحتياطي أو النقل لجهاز آخر

📥 استيراد الملاحظات
استيراد ملف JSON يحتوي على ملاحظات سابقة

دمج أو استبدال الملاحظات بسهولة

🗑️ إدارة كاملة للملاحظات
حذف ملاحظة واحدة

حذف جميع الملاحظات بضغطة زر

تأكيد قبل الحذف لحماية البيانات

التقنية	الاستخدام

🚀 كيفية التشغيل
قم بتحميل المشروع أو نسخه.

افتح ملف index.html في أي متصفح حديث.

ابدأ بإضافة ملاحظاتك فورًا — لا حاجة لأي إعدادات إضافية.

🔐 الأمان
لا يتم استخدام أي innerHTML لتجنب أي ثغرات XSS.

جميع النصوص تُعرض باستخدام textContent.

لا يتم إرسال أي بيانات إلى الإنترنت — كل شيء محلي بالكامل.
