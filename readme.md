📒 Notes App
A modern, lightweight web application for managing notes, crafted with a professional aesthetic and running entirely in the browser with no backend required.
It offers a smooth writing experience, automatic saving, instant search, dark mode support, and full import/export functionality.

✨ Features
📝 Create & Edit Notes
Easily add new notes

Edit existing notes at any time

Automatic saving using LocalStorage

Enter shortcut for quick saving

🔍 Instant Search
Search through titles, content, and categories

Results update instantly as you type

🎨 Beautiful & Responsive Design
Modern interface with soft, elegant colors

Fully responsive on phones and tablets

Smooth animations and subtle transitions for a refined experience

🌙 Dark Mode
Instant switching between light and dark themes

User preference saved in LocalStorage

📤 Export Notes
Export all notes as a JSON file

Ideal for backups or transferring notes to another device

📥 Import Notes
Import a valid JSON file containing previously saved notes

Easily merge or replace existing notes

🗑️ Full Notes Management
Delete individual notes

Delete all notes with a single click

Confirmation prompts to prevent accidental data loss

🧱 Technologies Used
Technology	Purpose
HTML5	Building the structure of the application
CSS3	Creating a beautiful and responsive UI
JavaScript (Vanilla)	App logic, storage, search, import/export
LocalStorage	Saving data locally without a server
📂 Project Structure
Code
project/

The app runs immediately by opening index.html in any modern browser.

🚀 How to Run
Download or clone the project

Open index.html in your browser

Start adding your notes instantly — no setup required

🔐 Security
No use of innerHTML to avoid XSS vulnerabilities

All text is rendered using textContent

No data is sent to the internet — everything is stored locally

📦 Import & Export
Export Notes
Click the 📤 Export button

A file named notes.json will be downloaded

Import Notes
Click the 📥 Import button

Select a valid JSON file

Notes will be loaded immediately

🌓 Dark Mode
Toggle between 🌙 / ☀️ with one click

Theme preference is saved automatically

Works even after reopening the browser
